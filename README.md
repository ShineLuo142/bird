# 一个基于gulp + scss + jq的电商网站demo
> 主要利用gulp + node 进行工程化开发，实现热加载，自动刷新，sass热编译。 
### 第一步 `git clone https://github.com/shineluohandsame/bird.git`
### 第二步 `cnpm i`前提条件电脑有node环境，使用该指令来安装gulp自动化所需要的所有依赖
### 第三步 `gulp dev`启动开发环境。
